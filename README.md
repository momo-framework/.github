# Momo Framework

> **Build with PHP, Run like Go.** > Скорость разработки PHP × Производительность C-расширений.

Momo — это современный PHP-фреймворк, спроектированный по принципу **Event-Driven Modular Monolith**. Он позволяет создавать сложные enterprise-системы (например, E-commerce), которые легко масштабируются из монолита в микросервисы без переписывания кодовой базы.
